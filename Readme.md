An api to make comination of an array of string  :



```
import requests

# now we make the test
login_data = {
    "list": ["a", "b", "c","d"],
    "separator": ";",
}
r = requests.post("https://young-earth-75813.herokuapp.com/api/v1/combine", json=login_data)
print(r.status_code)
print(r.content)
```
